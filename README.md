# WebRTC-SDK Demo

工程模板：Vue 3 + TypeScript + Vite  

用于演示 WebRTC-SDK 的使用方法。同时也用于测试 WebRTC-SDK 的功能。
## 项目运行
```bash
npm run dev
```
## 项目打包
```bash
npm run build
```
## 项目发布
发布至192.168.12.13服务器  

服务器ssh web web123456

服务静态资源目录 /home/webRTC/dist

## 注意事项
每次发布前必须修改公告牌，公告牌组件在`src/components/Billboard.vue`中。
